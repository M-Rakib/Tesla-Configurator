# Tesla Configurator (Model Y)

A Tesla-style **Model Y configurator** built with **HTML, CSS, and JavaScript**.  
Users can customise exterior/interior options, wheels, upgrades, and accessories while seeing the **total price** and an **estimated monthly payment** update in real time.

## Features
- Exterior colour selector (updates the exterior image)
- Interior colour selector (updates the interior image)
- Wheel selection (Standard vs Performance)
- Optional upgrades:
  - Full Self-Driving checkbox
  - Performance upgrade button
- Accessories selection (checkboxes)
- Live pricing + estimated payment breakdown

## Tech Stack
- HTML / CSS / JavaScript
- Tailwind CSS (CDN)
- Font Awesome (CDN)
